# NLP Sentiment Analysis using LSTM

This project performs binary sentiment analysis on text data using PyTorch.

## Task
Classify text as:
- Positive (1)
- Negative (0)

## Concepts Covered
- Text preprocessing
- Tokenization
- Vocabulary building
- Word embeddings
- LSTM for NLP
- Accuracy evaluation

## How to Run
```bash
python sentiment_lstm.py
