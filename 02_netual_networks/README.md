# Neural Networks (MLP)

This folder covers the implementation of a Multi-Layer Perceptron (MLP) using PyTorch.

## Covered
- Forward propagation
- Loss computation
- Backpropagation
- Training loop
- MLP using PyTorch nn.Module

## Files
- mlp_from_scratch.py → Manual MLP implementation
- mlp_using_nn_module.py → Clean nn.Module-based MLP

## How to Run
python mlp_from_scratch.py
python mlp_using_nn_module.py
