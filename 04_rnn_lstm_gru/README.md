# RNN, LSTM, and GRU Models

This folder contains sequence models implemented using PyTorch.

## Models Implemented
- Vanilla RNN
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)

## Task
Binary classification on synthetic sequential data.

## Concepts Covered
- Sequential data processing
- Hidden states
- Vanishing gradient problem
- Comparison of RNN vs LSTM vs GRU

## How to Run
python rnn_model.py
python lstm_model.py
python gru_model.py
